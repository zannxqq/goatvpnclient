# GoatWeb VPN Client 🐐

Современный, легковесный и мощный VPN-клиент под Windows, работающий на базе **Xray-core**. Поддерживает все продвинутые функции обхода блокировок.

![Screenshots]() <!-- Замени на ссылку на скриншот, когда зальешь его на GitHub -->

## 🚀 Основные возможности

* **Нативное ядро Xray-core:** Поддержка VLESS, Trojan, XTLS-Reality, TCP/WebSocket.
* **Режим TUN (Wintun):** Проксирование всего системного трафика на уровне сетевого интерфейса.
* **Интеграция** 
  * Поддержка профилей маршрутизации (Routing) и автообновления (Autorouting).
  * Парсинг полных Xray JSON-конфигов прямо из подписки.
  * Deep Links (`goatvpn://...`) для мгновенного импорта и управления.
  * Генерация и отправка аппаратного ключа (HWID).
* **Умный роутинг:** Разделение трафика (Proxy/Direct/Block) и защита от DPI (с помощью фрагментации и SNI-маскировки).
* **Стильный UI:** Fluent-дизайн с плавными анимациями, темной темой и поддержкой системного трея.

## 🛠 Технологический стек

* **Платформа:** WPF / .NET 9
* **Архитектура:** MVVM
* **VPN Движок:** Xray-core
* **Сетевой интерфейс:** Wintun

## 📥 Установка и запуск

Для обычных пользователей доступна готовая портативная версия:
1. Перейдите в раздел [Releases](../../releases).
2. Скачайте последний `.exe` файл.
3. Запустите (приложение само скачает актуальное ядро `xray-core` и базы `geoip/geosite` при первом запуске).

## 💻 Сборка из исходников (Для разработчиков)

Убедитесь, что у вас установлен [.NET 9 SDK](https://dotnet.microsoft.com/download/dotnet/9.0).

```bash
# Клонирование репозитория
git clone [https://github.com/ВАШ_НИК/GoatWebVPN.git](https://github.com/ВАШ_НИК/GoatWebVPN.git)
cd GoatWebVPN

# Запуск в режиме разработки
dotnet run

# Публикация в один EXE-файл
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
